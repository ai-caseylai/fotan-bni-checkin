# 後台管理手冊

## 登入

1. 開啟 **https://fotan.techforliving.net/admin**
2. 輸入密碼（預設 `admin888`）
3. 登入後進入總覽頁

![登入頁](../screenshots/admin-login.png)

## 導航

左側 sidebar 有 8 個頁面：

![Sidebar](../screenshots/admin-sidebar.png)

---

## 總覽

- **統計卡片**：總會議、會員人數、出席記錄、已付款
- **圖表**：出席趨勢折線圖 + 付款分佈甜甜圈圖
- **會議列表**：點擊展開查看出席詳情
- **CSV 匯出**：摘要 / 詳細兩種格式

![總覽](../screenshots/admin-overview.png)

---

## 簽到操作

### 選擇會議
下拉選單選擇會議，按「開始簽到」

### 操作
- **🕐 設定時間** — 簽到時間
- **💰 標記已付** — 付款狀態三級循環：未付 → 已付 → 免費 → 未付
- **✕ 標記缺席** — 標記為缺席
- **💬 催款** — 列出未付款者，一鍵複製 WhatsApp 連結
- **枱號** — 點擊人物可設定該次會議枱號

### 檢視模式
卡片模式 / 表格模式切換

![簽到操作](../screenshots/admin-checkin.png)

---

## 會議管理

### 列表模式
顯示會議統計：
- 會員人數 / 來賓人數 / 已收 / 未收

### 展開詳情
點擊 ▶ 展開出席名單（A-Z 排序），支援篩選：
- 📋 全部 / ✅ 已繳費 / ❌ 未繳費 / ✕ 缺席

### 日曆模式
月曆檢視，點擊會議日期跳轉簽到

### 新增會議
填寫日期、類型、收款人、來賓費
- ✅ **複製上次會議的與會會員** — 自動帶入上次出席會員

![會議管理](../screenshots/admin-meetings.png)

---

## 會員管理

- 75 位會員，卡片/表格雙模式
- 編輯：名稱、電話、電郵、專業、角色、會費日、簡介
- **付款憑證** — 上傳收據截圖（支援多張）
- 點擊 ▶ 查看出席歷史

![會員管理](../screenshots/admin-members.png)

---

## 來賓管理

- 63 位來賓（6/20 聚餐）
- 卡片模式自動載入出席記錄（✅/❌ 標示）
- 編輯時可指定所屬聚會、邀請人

---

## 系統設定

分為多個區塊：

| 區塊 | 內容 |
|------|------|
| ⚙️ 基本設定 | 頁面標題、載入文字 |
| ✏️ 簽到流程 | 簽到標題、確認文字、**跳過按鈕開關** |
| 🏷️ 狀態標籤 | 已付/未付/已簽到等標籤文字 |
| 🍽️ 午餐與節目 | 午餐費、枱號、時間表、主席話 |
| 🏅 委員 & 火炭會 | 委員介紹、火炭會介紹、申請入會連結 |
| 💳 付款連結 | PayMe/Alipay/FPS 連結、QR 碼上傳 |
| 🔐 密碼 | 修改管理密碼（需舊密碼） |
| 💿 備份 | 一鍵下載全資料庫 JSON |

![系統設定](../screenshots/admin-settings.png)
