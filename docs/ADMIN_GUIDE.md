## 總覽

- **統計卡片**：總會議、會員人數、出席記錄、已付款
- **圖表**：出席趨勢折線圖 + 付款分佈甜甜圈圖（Chart.js）
- **會議列表**：點擊展開查看出席詳情、A-Z 排序、付款篩選
- **CSV 匯出**：摘要 / 詳細兩種格式

## 簽到操作

### 選擇會議
下拉選單選擇會議

### 操作
- **🕐 設定時間** — 簽到時間
- **💰 標記已付** — 付款狀態三級：未付 → 已付 → 免費 → 未付
- **✕ 標記缺席** — 標記為缺席
- **枱號** — 點擊人物可設定該次會議枱號

### 付款操作模態頁
點擊出席者 → 彈出付款操作頁面：
- 📤 **憑證付費** — 上傳付款截圖
- 🆓 **免費嘉賓** — 標記為免費
- 💵 **現金付款** — 標記現金已付
- 🗑️ **移除此出席記錄**
- 📄 **已上傳憑證** — 顯示已上傳的收據圖片

> 📸 截圖：簽到操作頁面 + 付款模態頁

## 會議管理

### 列表模式
顯示會議統計：會員人數 / 來賓人數 / 已收 / 未收

### 展開詳情
點擊 ▶ 展開出席名單（A-Z 排序），支援篩選：
- 📋 全部 / ✅ 已繳費 / ❌ 未繳費 / ✕ 缺席

每人顯示：類型、姓名、簽到時間、付款狀態 badge（💰已付/🆓免費/❌💰未付）

### 日曆模式
月曆檢視會議日期

### 新增會議
填寫日期、類型、收款人、來賓費
- ✅ **複製上次會議的與會會員** — 自動帶入上次出席會員

> 📸 截圖：會議列表展開 + 新增會議表單

## 系統設定

分為多個區塊：

| 區塊 | 內容 |
|------|------|
| ⚙️ 基本設定 | 頁面標題、載入文字 |
| ✏️ 簽到流程 | 簽到標題、確認文字、**跳過按鈕開關** |
| 🏷️ 狀態標籤 | 已付/未付/已簽到等標籤文字 |
| 🍽️ 午餐與節目 | 午餐費、枱號、時間表、主席話 |
| 🏅 委員 & 火炭會 | 委員介紹、火炭會介紹、申請入會連結 |
| 💳 付款連結 | PayMe/Alipay/FPS 連結、QR 碼上傳 |
| 🔐 密碼 | 修改管理密碼（需舊密碼） |
| 💿 備份 | 一鍵下載全資料庫 JSON |

## 火炭會 Skill

- 📥 **下載 Skill 檔案** — 供 OpenClaw / Claude Code 使用
- 🔑 **Token 管理** — 新增 Token 時自動彈出 curl 範例（已填好 token）
- Token 有效期 90 天

### Skill REST API（/api/skill）

Token 保護，支援 9 種操作，唔需要 Node.js/Wrangler：

| Action | 用途 |
|--------|------|
| `import_guests` | 批次匯入嘉賓 |
| `update_payment` | 更新付款狀態 |
| `update_table` | 更新枱號 |
| `mark_arrival` | 標記出席/缺席 |
| `search` | 搜尋會員+來賓 |
| `meeting_stats` | 會議統計 |
| `payment_summary` | 付款摘要 |
| `list_meetings` | 會議列表 |
| `list_attendance` | 出席名單 |

curl 範例：
```bash
curl -s -X POST "https://fotan.techforliving.net/api/skill" \
  -H "Content-Type: application/json" \
  -d '{"token":"lob_xxxx","action":"import_guests","guests":[{"name":"陳大文","payment":"paid"}]}'
```
