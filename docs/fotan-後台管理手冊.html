登入頁

總覽

總覽

只顯示當前會議關鍵數據：

  • 📅 會議日期 + 類型
  • 👥 出席人數（總數、會員、來賓、已簽到）
  • 💰 收款（收費人數、免費人數、未收人數、收費標準、已收收入）
  • 📊 付款分佈甜甜圈圖（已收/免費/未收）
  • 📋 收入摘要（會員/來賓/收費/免費/未收明細）

> v3.7：已收人數分拆為「💰 收費」和「🆓 免費」兩張獨立卡片，營收只計算已付費者（不含免費嘉賓）。

簽到操作

簽到操作

  • 點擊人物卡片 → 自動記錄當前時間 + 彈出操作模態頁
  • 點擊付款 badge → 付款模態頁（💵現金 / 📤憑證付費 / 🆓免費）
  • ✕ 按鈕 → 標記缺席

付款模態

會議管理

會議管理

  • 列表/日曆雙模式
  • 5 個收費欄位：委員價、會員價、來賓價、早鳥價、臨場價
  • 展開出席名單（A-Z），支援付款篩選
  • 新增會議可複製上次會員

會員管理

會員管理

  • 26 個活躍會員，卡片/表格雙模式
  • 編輯：名稱、電話、電郵、專業、角色、會費日、簡介
  • 付款憑證上傳（支援多張）
  • 🆕 卡片直接顯示憑證縮圖（最多 3 張，超出顯示 +N）
  • 出席歷史

來賓管理

來賓管理

  • 表格模式含付款狀態欄位（💰已付 / 🆓免費 / ❌💰未付）
  • 卡片模式顯示出席記錄
