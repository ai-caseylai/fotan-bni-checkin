# 使用手冊 — 嘉賓簽到主頁

## 開啟主頁

手機或電腦開啟 **https://fotan.techforliving.net**

> 📸 截圖：主頁頂部標題 + 枱號顯示

## 簽到流程

### 1. 選擇嘉賓
底部 tab bar 預設顯示「未簽到」名單，所有人按 A-Z 排列。每人顯示：姓名、會員/來賓標籤、付款狀態（已付款/未付款）、枱號。

> 📸 截圖：未簽到名單

### 2. 未付款 → 直接跳到付款頁
點擊未付款嘉賓 → 直接顯示付款頁面，提供三種付款方式：
- **PayMe** — 點擊直接跳轉 PayMe 付款
- **Alipay** — 顯示 QR Code
- **FPS 轉數快** — 顯示 QR Code
- **上傳付款憑證** — 選擇截圖上傳

> 📸 截圖：付款頁面（PayMe/Alipay/FPS 連結 + 憑證上傳）

### 3. 已付款 → 確認簽到
點擊已付款嘉賓 → 彈出確認對話框 → 按「✓ 確認」→ 播放簽到音效

> 📸 截圖：確認簽到對話框

### 4. 完成簽到
簽到完成後顯示「多謝光臨」及枱號、節目表、主席的話。

> 📸 截圖：簽到完成頁面

## 底部 Tab Bar

| Tab | 功能 |
|-----|------|
| 👥 未簽到 | 顯示尚未簽到的人（按 A-Z 排列） |
| ✅ 已簽到 | 已簽到名單，點擊查看個人資料卡 + vCard |
| 📋 流程 | 節目時間表 + 主席的話 |
| 🏅 委員 | 委員介紹 |
| ℹ️ 關於 | 火炭會介紹 + 申請入會連結 |

## 個人資料卡

在「已簽到」tab 點擊任何人，彈出資料卡：
- 頭像、姓名、枱號
- 💼 專業、🏅 角色
- 📱 電話（點擊撥打）
- 📧 電郵
- 📇 **加入通訊錄 (vCard)** — 下載 .vcf 直接加入手機電話簿

> 📸 截圖：個人資料卡

## 枱號

每個人名字右側顯示枱號（如 `3號枱`），由後台管理員設定。每枱最多 12 人。

## 付款狀態顯示

| 顯示 | 意思 |
|------|------|
| 已付款 | 已成功付款 |
| 未付款 | 尚未付款，點擊跳到付款頁 |
