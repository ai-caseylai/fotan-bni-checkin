# 使用手冊 — 嘉賓簽到主頁

## 開啟主頁

手機或電腦開啟 **https://fotan.techforliving.net**

![主頁](../screenshots/main-page.png)
*（截圖：主頁頂部標題 + 枱號顯示）*

## 簽到流程

### 1. 選擇嘉賓
底部 tab bar 預設顯示「未簽到」名單，所有人按 A-Z 排列。

![未簽到](../screenshots/unchecked-tab.png)

### 2. 點擊簽到
點擊人名 → 彈出確認對話框 → 按「✓ 確認」

![確認簽到](../screenshots/confirm-checkin.png)

### 3. 付款
簽到後如未付款，會顯示付款頁面：

- **PayMe** — 點擊直接跳轉 PayMe 付款
- **Alipay** — 顯示 QR Code
- **FPS 轉數快** — 顯示 QR Code
- **上傳付款憑證** — 選擇截圖上傳
- **跳過，直接簽到** — 不付款直接完成簽到（可由後台關閉）

![付款頁](../screenshots/payment-page.png)

### 4. 完成簽到
付款後或上傳憑證後，顯示「多謝光臨」及枱號。

![完成](../screenshots/checkin-done.png)

## 底部 Tab Bar

| Tab | 功能 |
|-----|------|
| 👥 未簽到 | 顯示尚未簽到的人 |
| ✅ 已簽到 | 已簽到名單，點擊查看個人資料卡 + vCard |
| 📋 流程 | 節目時間表 + 主席的話 |
| 🏅 委員 | 委員介紹 |
| ℹ️ 關於 | 火炭會介紹 + 申請入會連結 |

## 個人資料卡

在「已簽到」tab 點擊任何人，彈出資料卡：

![資料卡](../screenshots/person-card.png)

- 頭像、姓名、枱號
- 💼 專業、🏅 角色
- 📱 電話（點擊撥打）
- 📧 電郵
- 📇 **加入通訊錄 (vCard)** — 下載 .vcf 直接加入手機電話簿

## 枱號

每個人名字右側顯示枱號（如 `3號枱`），由後台管理員在簽到操作頁設定。
