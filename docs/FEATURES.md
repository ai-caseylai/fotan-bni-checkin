# 火炭會聚會簽到系統 — 銷售功能手冊

## 系統簡介

**火炭會聚會簽到系統** 是一套專為商務聚會（BNI 例會、商會聚餐、行業交流會）設計的全流程數碼管理平台。從來賓報到、付款追蹤、餐桌排位到營收統計，一站式管理，大幅提升聚會營運效率。

🌐 **網址**：`fotan.techforliving.net`
📱 **平台**：手機/平板/桌面全適配 PWA，可安裝到主畫面

## 白話說明：呢個系統幫到你啲乜？

### 🎯 一句講晒

**一部手機搞掂成個聚會 — 來賓自己簽到、自己畀錢、系統自動排位、自動計數。**

---

### 🚪 門口簽到 — 唔使再排隊寫名

以前：來賓到場要排隊、填紙仔、人手搵名、mark 低幾點到。10 個人排 5 分鐘。

而家：來賓拎出手機掃 QR Code → 見到自己個名 → 㩒一下 → 搞掂。**3 秒簽完一個。**

---

### 💵 收錢 — 唔使再追數

以前：財政要逐個問「你畀咗錢未？」、mark 低邊個現金邊個 PayMe、散紙找來找去、埋數對唔齊。

而家：
- 來賓簽到時見到**付款掣**（PayMe / Alipay / FPS），㩒一下就跳去畀錢
- 畀完錢**影張相**（cap 圖）上傳，系統自動標記「已付款」
- 財政打開手機一睇：**邊個畀咗、邊個未畀、總共收咗幾多** — 全部即時計好

---

### 🍽 排位 — 唔使再畫紙仔

以前：秘書用 Excel 畫枱、print 出嚟、改來改去、當日有人 no show 亂晒。

而家：
- 手機/電腦**拖放排位**，拉個名去張枱就搞掂
- 想 VIP 全部坐一枱？㩒一個掣「VIP 自動排位」— **1 秒排好**
- 臨時有人唔嚟？拉走佢就得，全枱自動調整
- 排完位即時喺簽到頁面顯示，來賓簽到時見到自己坐邊枱

---

### 📊 睇數 — 唔使再用人手計

以前：主席問「今日幾多人？收咗幾多錢？」財政要㩒計數機計 10 分鐘。

而家：打開手機 → **總覽頁面** → 即時顯示：
- 出席人數（委員幾多、會員幾多、來賓幾多、簽咗到幾多）
- 收款狀況（已收幾多、仲爭幾多）
- 營收總額

---

### 🤖 龍蝦仔 AI 助手 — 問就有答案

唔想自己㩒來㩒去？**用口講就得**：

> 「今日有幾多人出席？」「幫陳大文簽到」「VIP 全部坐 1 號枱」「邊個未畀錢？」

龍蝦仔會自動幫你搞掂，**講廣東話就得**。

---

### 📱 來賓手機畫面

來賓打開網址，見到：
- **自己個名** + 行業 + 枱號
- **付款掣**（PayMe / Alipay / FPS）
- **今日節目表**（幾點開始、幾點食飯）
- **主席歡迎詞**

成個過程唔使裝 App，唔使 register，就咁開網頁就得。

## 核心功能一覽

### ✅ 極速簽到
- QR Code 一掃即簽，3 秒完成
- 自動分配枱號 + 座位號
- 簽到成功顯示枱名、節目表、主席歡迎詞
- Cookie 記住簽到狀態，避免重複簽到
- 支援「直接簽到」模式（跳過付款檢查）

### 💰 多元付款追蹤
- **6 種付款方式**：現金、PayMe、Alipay HK、FPS 轉數快、銀行轉帳、收據上傳
- Deep link 一鍵跳轉付款 App
- 來賓自助上傳付款收據，系統自動標記已付款
- 5 級費用制：委員、會員、來賓、早鳥、現場（每場會議可獨立設定）
- 付款狀態：已付/免費/未付，一目了然

### 🍽 智能餐桌排位
- 拖放式視覺排位畫布（自由位置 or 格線模式）
- 自訂枱名（VIP 枱、主家席、嘉賓枱...）
- 枱數 1-20 自由配置，每枱人數上限自訂
- **一鍵自動排位**：按委員/貴賓/會員/來賓/標籤/姓氏分組
- 整枱搬人（含容量檢查 + 強制溢位）
- 排位即時同步至簽到頁面

### 📊 即時統計儀表板
- 出席人數（總數/委員/會員/貴賓來賓/一般來賓）
- 簽到進度（已簽到/未簽到/缺席）
- 付款分佈（已付/免費/未付）Chart.js 圖表
- 營收總計 + 預期收入
- 支援 CSV 匯出

### 🤖 AI 智能助手「龍蝦仔」🦞
- 粵語對話，自然語言操作
- 支援 32 項 Function Calling（查詢、簽到、排位、付款、統計）
- 圖片分析（上傳 QR Code 截圖自動辨識）
- 真正的 AI Agent — 說「幫我排位」即自動執行

### 👥 完整人員管理
- 會員管理（角色、標籤、會費記錄、收據上傳）
- 來賓管理（VIP 標記、邀請人、出席歷史）
- 批次匯入（Excel/CSV → 一鍵匯入）
- 跨會議出席歷史追蹤
- 搜尋（姓名/電話/行業/職位）

### 📅 多會議支援
- 每場會議獨立管理（日期、類型、費用）
- 複製上期會議出席名單到新會議
- 會議類型：例會/特別會議/週年慶
- 會議統計獨立計算

### 📱 行動簽到頁面
- 6 個底部分頁：未簽到 / 已簽到 / 節目表 / 遊戲 / 委員介紹 / 相簿
- 搜尋欄（姓名/電話/行業/職位即時過濾）
- 人物卡片 popup（姓名、角色、行業、電話、email、枱號、vCard 下載）
- 主席歡迎訊息
- 節目流程表

### 🔔 通訊整合
- **WhatsApp 提醒**：一鍵發送個人化提醒（含付款連結）
- **Telegram Bot**：即時通知簽到動態
- 自訂訊息範本

### 🔒 安全與權限
- 管理後台密碼保護（bcrypt 加密 + IP 速率限制）
- API Token 制（90 天有效期，可撤銷）
- Cookie HttpOnly + SameSite=Strict Session

## 為什麼選擇我們？

1. **極低成本** — Cloudflare 免費額度內運行，零伺服器費用
2. **無需安裝** — 瀏覽器即開即用，PWA 可安裝
3. **AI 原生** — 內建粵語助手，語音/文字操作
4. **全流程度** — 簽到→付款→排位→統計，一站式
5. **行動優先** — 手機體驗完美，QR Code 一掃搞定
6. **開放 API** — 31 個 Skill Action，可整合任何第三方系統
